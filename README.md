# DealFinder
A python program to scrape various websites and alert the user when a game drops below a willingness to pay

To setup run GameScraper.py with command line argument -s, before actually scraping the sites the program requires you to initialize 
and add games in setup mode with i and a commands respectively. 

When userdata and game links are set you can run without command line argument to scrape the sites, you should receive an email if the 
game is available under your set willingness to pay at any of the sites
